I began my professional journey at DgHub, a fintech company, where I built and optimized core backend systems for digital wallet applications. My responsibilities included developing robust APIs for a mobile wallet, building an MVC platform for merchants, and integrating third-party SOAP APIs for payment services. Additionally, I contributed to modernizing our ecosystem by migrating legacy ASP.NET applications to .NET 6.

This role deeply immersed me in enterprise software design, giving me hands-on experience with digital ledgers, transaction management, system scaling, and disaster recovery, while teaching me how software successfully transitions from local code to production.

## Highlights

- Developed and maintained high-performance fintech web applications using ASP.NET MVC, .NET 6, C#, and SQL Server.

- Integrated secure third-party payment services and SOAP APIs to ensure reliable, encrypted transaction processing.

- Accelerated legacy system modernization by actively migrating applications to .NET 6, enhancing overall performance and maintainability.

- Implemented industry-standard API practices, including versioning, rate limiting, data encryption, and comprehensive Postman testing workflows.
